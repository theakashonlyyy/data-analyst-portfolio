# Sales Data Analysis using Python & SQL

## Project Overview
This project focuses on analyzing sales data to identify trends, patterns, and business insights using Python and SQL.  
The goal is to help businesses understand their sales performance and make data-driven decisions.

## Tools & Technologies
- Python (Pandas, NumPy, Matplotlib, Seaborn)
- SQL
- Jupyter Notebook
- Excel (for initial data review)

## Dataset
- Sales dataset containing information about orders, customers, products, revenue, and dates.
- Data is cleaned and processed before analysis.

## Key Analysis Performed
- Data cleaning and preprocessing
- Monthly and yearly sales trends
- Top-selling products analysis
- Revenue analysis by category
- Customer purchase behavior

## SQL Analysis
- SELECT, WHERE, GROUP BY, ORDER BY
- Aggregations (SUM, AVG, COUNT)
- Business-driven queries for insights

## Python Analysis
- Data manipulation using Pandas
- Exploratory Data Analysis (EDA)
- Data visualization using Matplotlib & Seaborn

## Insights & Findings
- Identified peak sales months
- Found top revenue-generating products
- Observed customer buying patterns

## Conclusion
This project demonstrates practical data analysis skills using Python and SQL and reflects real-world business problem-solving.
